# あなたのゲーム内チャット倉庫 (In-Game Chat Vault)

ゲーム内チャットで使用する定型文を効率よく管理・コピーするためのWebツールです。
サーバーへの送信を行わず、すべてブラウザ内のローカルストレージで完結するため、安全かつ高速に利用できます。
ただ定型文をコピーできるだけのサイトなので、ゲーム以外でも使えるかも。

## 🚀 主な機能

- **ワンクリックコピー**: 登録したフレーズをクリックするだけでクリップボードにコピー。
- **ドラッグ＆ドロップ**: カテゴリ間での自由な並び替え。
- **自動保存・履歴復元**: 操作ごとに最大20件まで自動保存。「戻す」「進む」機能に加え、「履歴から復元」機能から以前の状態へ復元可能。
- **レスポンシブデザイン**: PCのほか、スマートフォンやタブレットのブラウザにも最適化。
※「戻す」「進む」機能は、画面を閉じると履歴が消去され、使えなくなります。「履歴から復元」機能は保持されているので、
　誤って全消去してからブラウザの更新等を行った際は、「履歴から復元」機能を使用してください。

## 🛠 使い方

1. あなたのゲーム内チャット倉庫をブラウザで開きます。
2. 編集アイコン（✎）から、カテゴリ名を変更して、使いやすいようにカスタマイズします。
3. ＋ボタンからカードを追加、定型文を入力します。
4. ゲーム中、必要な文をクリックしてコピーし、ゲーム内のチャット欄へ貼り付けて使用します。
